Bzf format specification

Type: Bzf

**BZF**: `BZF ` + `V1.0` header, then **LZMA** payload that expands to a normal **BIF** (`BIF.ksy`). Common on mobile KotOR ports.

OffsetSizeIDTypeNote
0 ... file_type str(ASCII) File type signature. Must be "BZF " for compressed BIF files.
4 ... version str(ASCII) File format version. Always "V1.0" for BZF files.
8 ... compressed_data LZMA-compressed BIF file data (single blob to EOF). Decompress with LZMA to obtain the standard BIF structure (see BIF.ksy).
