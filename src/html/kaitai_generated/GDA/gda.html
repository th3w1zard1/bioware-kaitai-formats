Gda format specification

Type: Gda

**GDA** (Dragon Age 2D array): **GFF4** stream with top-level FourCC **`G2DA`** and `type_version` `V0.1` / `V0.2` (`GDAFile::load` in xoreos). On-disk struct templates reuse imported **`gff::gff4_file`** from `formats/GFF/GFF.ksy`. G2DA column/row list field ids: `meta.xref.xoreos_gff4_g2da_fields`. Classic Aurora `.2da` binary: `formats/TwoDA/TwoDA.ksy`. **reone:** not applicable for GDA wire ingestion on the KotOR fork (`meta.xref.reone_gda_consumer_note`).

OffsetSizeIDTypeNote
0 ... as_gff4 Gff4File On-disk bytes are a full GFF4 stream. Runtime check: `file_type` should equal `G2DA` (fourCC `0x47324441` as read by `readUint32BE` in xoreos `Header::read`).
