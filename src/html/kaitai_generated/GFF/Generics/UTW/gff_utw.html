GffUtw format specification

Type: GffUtw

**UTW** resources are **GFF3** on disk (Aurora `GFF ` prefix + V3.x version). Wire layout is fully defined by `formats/GFF/GFF.ksy` and `formats/Common/bioware_gff_common.ksy`; this file is a **template capsule** for tooling, `meta.xref` anchors, and game-specific `doc` without duplicating the GFF3 grammar. FileType / restype id **2058** — see `bioware_type_ids::xoreos_file_type_id` enum member `utw`.
