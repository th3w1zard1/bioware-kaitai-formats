Da2s format specification

Type: Da2s

**DA2S** (Dragon Age 2 save): Eclipse binary save — `DA2S` signature, `version==1`, length-prefixed strings + tagged blocks (party/inventory/journal/etc.). **Not KotOR** — Andastra serializers under `Game/Games/Eclipse/DragonAge2/Save/` (`meta.xref`).

OffsetSizeIDTypeNote
0 ... signature str(ASCII) File signature. Must be "DA2S" for Dragon Age 2 save files.
4 ... version s4le Save format version. Must be 1 for Dragon Age 2.
8 ... save_name LengthPrefixedString User-entered save name displayed in UI
??? ... module_name LengthPrefixedString Current module resource name
??? ... area_name LengthPrefixedString Current area name for display
??? ... time_played_seconds s4le Total play time in seconds
??? ... timestamp_filetime s8le Save creation timestamp as Windows FILETIME (int64). Convert using DateTime.FromFileTime().
??? ... num_screenshot_data s4le Length of screenshot data in bytes (0 if no screenshot)
??? ... screenshot_data u1 Screenshot image data (typically TGA or DDS format)
??? ... num_portrait_data s4le Length of portrait data in bytes (0 if no portrait)
??? ... portrait_data u1 Portrait image data (typically TGA or DDS format)
??? ... player_name LengthPrefixedString Player character name
??? ... party_member_count s4le Number of party members (from PartyState)
??? ... player_level s4le Player character level (from PartyState.PlayerCharacter)
