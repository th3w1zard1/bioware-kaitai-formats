Wav format specification

Type: Wav

**KotOR WAV:** standard **RIFF/WAVE** (`fmt ` + `data`) plus engine-specific cases (VO vs SFX obfuscation wrappers, MP3-in-WAV quirks) described on the PyKotor wiki — this `.ksy` models the **core RIFF chunk tree**; 470-byte SFX / 20-byte VO prefixes are application-level. `wFormatTag` / PCM layout notes: `bioware_common.ksy` → `riff_wave_format_tag`.
