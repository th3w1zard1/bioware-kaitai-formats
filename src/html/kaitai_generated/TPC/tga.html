Tga format specification

Type: Tga

**TGA** (Truevision Targa): 18-byte header, optional color map, image id, then raw or RLE pixels. KotOR often converts authoring TGAs to **TPC** for shipping. Shared header enums: `formats/Common/tga_common.ksy`.

OffsetSizeIDTypeNote
0 ... id_length u1 Length of image ID field (0-255 bytes)
1 ... color_map_type u1→TgaCommon::TgaColorMapType Color map type (`u1`). Canonical: `formats/Common/tga_common.ksy` → `tga_color_map_type`.
2 ... image_type u1→TgaCommon::TgaImageType Image type / compression (`u1`). Canonical: `formats/Common/tga_common.ksy` → `tga_image_type`.
3 ... color_map_spec ColorMapSpecification Color map specification (only present if color_map_type == present)
8 ... image_spec ImageSpecification Image specification (dimensions and pixel format)
18 ... image_id str(ASCII) Image identification field (optional ASCII string)
??? ... color_map_data u1 Color map data (palette entries)
??? ... image_data u1 Image pixel data (raw or RLE-compressed). Size depends on image dimensions, pixel format, and compression. For uncompressed formats: width × height × bytes_per_pixel For RLE formats: Variable size depending on compression ratio

Type: ColorMapSpecification

OffsetSizeIDTypeNote
0 ... first_entry_index u2le Index of first color map entry
2 ... length u2le Number of color map entries
4 ... entry_size u1 Size of each color map entry in bits (15, 16, 24, or 32)

Type: ImageSpecification

OffsetSizeIDTypeNote
0 ... x_origin u2le X coordinate of lower-left corner of image
2 ... y_origin u2le Y coordinate of lower-left corner of image
4 ... width u2le Image width in pixels
6 ... height u2le Image height in pixels
8 ... pixel_depth u1 Bits per pixel: - 8 = Greyscale or indexed - 16 = RGB 5-5-5 or RGBA 1-5-5-5 - 24 = RGB - 32 = RGBA
9 ... image_descriptor u1 Image descriptor byte: - Bits 0-3: Number of attribute bits per pixel (alpha channel) - Bit 4: Reserved - Bit 5: Screen origin (0 = bottom-left, 1 = top-left) - Bits 6-7: Interleaving (usually 0)
